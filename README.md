# Laravel POS System

A complete Point of Sale (POS) system built with Laravel featuring invoice management, customer management, and FBR Digital Invoicing integration.

## Features

- Product Management
- Customer Management
- Invoice Generation
- FBR Digital Invoicing
- Sales Reports
- Admin Dashboard
- Role & Permission Management
- REST APIs

## Tech Stack

- Laravel
- PHP
- MySQL
- Bootstrap
- JavaScript

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
